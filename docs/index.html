Resume Parsing

Extracts data from PDF resumes into specific fields (Education, Work, Skills) for precise, consistent reuse.

Smart Autofill

Eliminate repetitive data errors. Our intelligent engine fills portals (LinkedIn, Greenhouse, Lever) with your curated, best-version data.

Manual Entry

Provides a sidebar with categorized, clickable data for fields that require manual input.

Local Storage

All data is stored locally in your browser. No external servers, no data collection.

Installation

  1. Download the extension:
    Click to Download applyly.zip
  2. Unzip/Extract the downloaded file. You should see a dist folder.
  3. Open Chrome and go to chrome://extensions/.
  4. Toggle Developer mode (top right corner).
  5. Click Load unpacked.
  6. Select the extracted dist folder.
