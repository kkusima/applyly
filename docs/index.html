How it works

1. Extract

Upload your resume to create a local profile. Applyly automatically parses and structures your data.

2. Configure

Review and refine your information. Create multiple profiles for different roles or industries.

3. Apply

Navigate to any job application portal. Use Applyly to instantly fill repetitive fields with one click.

Installation

  1. Clone the repository from GitHub.
  2. Open Chrome and go to chrome://extensions/.
  3. Enable Developer mode.
  4. Click Load unpacked and select the dist directory.

Open Source

Applyly is built for the community. It is free, open-source, and privacy-focused. No analytics, no tracking, no data leaves your browser.
